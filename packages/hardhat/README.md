# MockG Token

This is a simple mock implementation of goodDollar token for testing purposes 
on the Celo Sepolia testnet.

## Deployment

The MockG token has been deployed to Sepolia at:
0x89993fbd7F8EaCD95BE9734dFA99Da506693eD23

The owner (deployer) has 100,000,000,000 tokens with 18 decimals.

## Available Scripts

- `yarn deploy-mock-g` - Deploy a new MockG token to Celo Sepolia
- `yarn verify-mock-g` - Verify the deployed MockG token
- `yarn contract-verification` - Source code verification in Blockscout

## Token Details

- Name: Mock G
- Symbol: MG
- Decimals: 18
